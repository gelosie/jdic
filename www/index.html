The JDesktop Integration Components (JDIC) project aims to make Java™ technology-based applications ("Java applications") first-class citizens of current desktop platforms without sacrificing platform independence. Its mission is to enable seamless desktop/Java integration.

JDIC provides Java applications with access to functionalities and facilities provided by the native desktop. It consists of a collection of Java packages and tools. JDIC supports a variety of features such as embedding the native browser, launching the desktop applications, creating tray icons on the desktop, registering file type associations, creating JNLP installer packages, etc. Many new features are contributed as incubator projects from the community.

See the initial JDIC Overview article for a more thorough read on the project. Check out George Zhang's Blog for more information. Read more articles relating to JDIC, with code examples and evaluations of the project.

Incubator Projects

Some projects are associated with JDIC but aren't part of the source or binary distribution of the JDIC project. These incubator projects are owned by the owners (listed in alphabetic order), released as separate downloads and develop at their own speed with different quality criteria. Once an incubator project complies with the requirements of JDIC, it might move into the JDIC project. Check out the Setup An Incubator Project page for detailed instructions.

Below incubator projects currently exist (in order of hosting date, from earliest to latest):

  • JDICplus SDK - Java WIN32-extension development kit. - New!
    Mainly the project implements the lightweight wrapper component around Microsoft Internet Explorer.
    Owner(s): uta
  • SaverBeans Screensaver SDK - Java screensaver development kit
    Owner(s): markroth8
  • Floating Dock Toplevel Window - Java API to develop a floating dock toplevel window
    Owner(s): bino_george, laszlo
  • Icon Service - Returns a Java icon object from a native icon specification
    Owner(s): Chas
  • SystemInfo - Checks the system information
    Owner(s): Carl Dea, John Ellis
  • Misc - Hosts those simple (one single method, one single class type of) APIs.
    Developer(s): Carl Dea, Joshua Marinacci
  • FileUtil - File utility API, extending java.io.File class
    Owner(s): Fábio C. Martins
  • Music Player Control API - Java API to control native music players.
    Owner(s): Zsombor Gegesy

Note: The initial incubator project Tray Icon API was moved to JDIC releases from release 0.8.6.

FileExplorer

A simple cross-platform file viewer that allows the users to browse the files and folders on local drives. It demonstrates functionality in the JDIC API's org.jdesktop.jdic.desktop package.

Right-clicking a selected file brings up a popup menu. The items in this menu — Open, Edit, Print, Browse, and Mail to... — correspond to methods in the org.jdesktop.jdic.desktop.Desktop class.

At the upper right of the GUI is a Browse button. Clicking it launches the default browser to display the item specified by the Address text field, which can be a local file path or a URL.

Browser

A simple cross-platform browser application that demonstrates functionality in the JDIC API's org.jdesktop.jdic.browser package. This demo embeds a WebBrowser component to render the HTML page specified in the URL text field. The top-left buttons allow the user to go forward and back and to reload and stop loading pages. The status bar reflects the browser's current state.

Note: To run this demo, you must use Internet Explorer or Mozilla (1.4 or a higher version) on Windows platforms; use Mozilla (1.4 or a higher version) on Linux/Unix platforms.

TrayIcon

This demo application demonstrates functionality in the JDIC APIs org.jdesktop.jdic.tray package. 

It creates a tray icon on the desktop (in the System Tray Area for Windows platforms, or in the Notification Area for Unix platforms), with a caption (text), an animated icon, and an associated Swing menu containing icons. It also has a tooltip displayed when the mouse hovers over the tray icon.

Troubleshooting Note:  
On Linux, if you do not see tray icons, the notification area is probably disabled. You can enable it by right-clicking the GNOME panel and selecting Add to Panel -> Utility -> Notification Area.

Floating Dock

This is a collection of demos about using the Floating Dock class to create Java applications attached to the edge of the screen. Follow the link on the left for full details.
WallPaper
 This demo application demonstrates the Wallpaper API, functionality in the JDIC org.jdesktop.jdic.misc package.

It displays a file chooser filtering the various popular image file formats to be selected as your desktop background. At the upper right, a combo box displays different styles that can be selected, which manipulate the image when the image is applied as the background.

Note: This demo currently runs on the Windows platform. When using Windows, without Internet Explorer 4.0 or above the background can only be set with an image file of type bitmap format.
